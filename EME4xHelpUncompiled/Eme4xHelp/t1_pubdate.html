EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Publication Date
Type: text
Short Name: pubdate

FGDC Definition

The date when the data set is published or otherwise made available for release.

EPA Requirement/Guidance

This field is required for EPA geospatial assets as per minimum FGDC standards. Follow FGDC guidance and information provided below for implementation of this field.

Domain

  • Unknown
  • Unpublished material
  • free date
Note: Date information posted within the publication date field should be documented using the standard date format (YYYY, YYYYMM, or YYYYMMDD, where Y=year, M=month, D=day [no slashes are permitted]), or it may be in the form of free text, such as 'Unpublished Material' or 'Unknown'.
