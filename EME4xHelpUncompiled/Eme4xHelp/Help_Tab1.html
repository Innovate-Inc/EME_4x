EPA Metadata Editor (EME)

Interface Overview and Element Description: Tab 1

Tab 1 of the EPA Metadata Editor allows a user to specify information that corresponds to all EPA required elements in Section 1 of the FGDC CSDGM.

Geospatial metadata elements made available for editing within Tab 1 of the EPA Metadata Editor are described in greater detail below.

Tab 1 Elements Overview:

  • Abstract
  • Access Constraints
  • Contact Information
  • Currency
  • East Bounding Coordinates
  • EPA Theme Keyword
  • ISO Theme Keyword
  • North Bounding Coordinates
  • Online Linkage
  • Origin
  • Progress
  • Publisher
  • Publication Date
  • Publication Place
  • Purpose
  • Security Classification
  • South Bounding Coordinates
  • Supplemental Information
  • Time Period Information
  • Title
  • Update Frequency
  • Use Constraints
  • West Bounding Coordinates
