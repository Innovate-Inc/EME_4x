EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Vertical Positional Accuracy Explanation
Type: text
Short Name: vertacce

FGDC Definition

The identification of the test that yielded the Vertical Positional Accuracy Value.

EPA Requirement/Guidance

This field is not required for minimum EPA compliancy. If data contain vertical information and tests are performed to yield an accuracy assessment for vertical information, this field should be used to document the test(s) performed.
