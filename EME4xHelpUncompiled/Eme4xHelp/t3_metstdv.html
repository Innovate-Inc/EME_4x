EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Metadata Standard Version
Type: text
Short Name: metstdv

FGDC Definition

Identification of the version of the metadata standard used to document the data set

EPA Requirement/Guidance

Required for EPA geospatial assets as per minimum FGDC standards. Typically, the value for this field should be 'FGDC-STD-001-1998' because the EPA implementation is based on this version. Follow FGDC guidance for implementation of this field.
