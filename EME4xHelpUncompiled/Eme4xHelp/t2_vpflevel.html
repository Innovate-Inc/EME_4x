EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: VPF Topology Level
Type: integer
Short Name: vpflevel

FGDC Definition

The completeness fo the topology carried by the data set. The levels of completeness are defined in Department of Defense, 1992, Vector Product Format (MIL-STD-600006): Philadelphia, Department of Defense, Defense Printing Service Detachment Office.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but becomes required when the parent element, "Point and Vector Object Information" is used.

Domain

0 < = VPF Topology Level < = 3
