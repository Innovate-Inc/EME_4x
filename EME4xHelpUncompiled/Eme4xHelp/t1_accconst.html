EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Access Constraints
Type: text
Short Name: accconst

FGDC Definition

Restrictions and legal prerequisites for accessing the data set. These include any access constraints applied to assure the protection of privacy or intellectual property, and any special restrictions or limitations on obtaining the data set.

EPA Requirement/Guidance

Provide information about access constraints for the asset to describe decisions made for applying security restrictions. For example, if data are classified as 'High Confidentiality' within the security classification field, include information in this field to identify reason(s) for restricting distribution.
