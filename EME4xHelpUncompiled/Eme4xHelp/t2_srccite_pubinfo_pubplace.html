EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Source Citation: Publication Place
Type: text
Short Name: pubplace

FGDC Definition

The name of the city (and state or province, and country, if needed to identify the city) where the data set was published or released.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but becomes required when the parent element, Publication Information, is used.
