EPA Metadata Editor (EME)

Interface Element Description: Tab 2 - Metadata Author

Mandatory element

EPA Requirement/Guidance

This element describes the responsible party for the metadata. The metadata author should be an individual who is knowledgeable about the metadata, and may be able to answer questions about the metadata record, as opposed to the data specifically (this may also be the same person as the data set point of contact).
