EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Metadata Author
Type: compound CI_ResponsibleParty
ISO Element Name: contact

EPA Requirement/Guidance

Provide detailed information of whom should be contacted regarding the metadata specifically. The metadata author should be an individual who is knowledgeable about the metadata, and may be able to answer questions about the metadata record, as opposed to the data specifically (this may also be the same person as the data set point of contact). Details on how to provide contact information for the data set are provided in Tab 2.
