EPA Metadata Editor (EME)

Interface Element Description: Tab 2 - Linkage

Optional element

EPA Requirement/Guidance

Linkage is not required for EPA minimum requirements; however, it is highly encouraged in order to promote reuse of and access to resources. Linkage can be found in the following elements:

  • Distribution > Contact Details - Provide a URL for any information about the resource distribution.
  • Distribution > Digital Transfer Options > Online Resource- Provide a URL for any information about where the resource can be obtained. In some cases, more than one digital transfer option is available for any given resource. Project Open Data regards Download URL and Web Service Endpoint as mandatory if applicable, while also requesting Data Dictionary and Homepage URL as optional elements. The order of these URLs is not important, only the function code.

    Download URL - Function Code: Download. URL providing direct access to the downloadable distribution of a dataset.

    Web Service - Function Code: webService. Endpoint of web service to access dataset. This is typically the REST endpoint for an ArcGIS Server service, but it may also be a WMS GetCapabilities URL or a SOAP WSDL endpoint.

    Data Dictionary - Function Code: information. URL for accessing related documents such as technical information about a dataset, developer documentation, etc.

    Homepage URL - Function Code: search. Alternative landing page used to redirect user to a contextual, Agency-hosted “homepage” for the Dataset or API when selecting this resource from a metadata catalog. This field is not intended for an agency’s homepage (e.g. www.agency.gov), but rather if a dataset has a human-friendly hub or landing page that users should be directed to for all resources tied to the dataset. This allows agencies to better specify what a visitor receives after selecting one of the agency’s datasets on Data.gov or in third-party mashups.

    Other Online Resources: Any other URLs related to this resource.

    Examples:

      • Online Resource 1 Linkage Download URL:
      "https://edg.epa.gov/data/Public/OEI/EPA_Facilities/EPA_Locations.zip"
      Online Resource 2 Linkage Map Service:
      "http://geodata.epa.gov/arcgis/rest/services/OEI/EPA_Locations/MapServer"

  • Contacts > Resource Owner Contact Details - Provide a URL for any information about the resource owner.
  • Contacts > Resource Producer Contact Details - Provide a URL for any information about the resource producer.
  • Contacts > Metadata Author Contact Details - Provide a URL for any information about the metadata author.

Note - ArcCatalog often populates this field with a local path to data, or the details of an ArcSDE connection. This conflicts with the FGDC and ISO 19115 guidance as well as Data.gov requirements. If this occurs, replace the field content with a URL.

Domain

Linkage Type:

  • download - online instructions for transferrring data from one storage device or system to another
  • information - online information about the dataset
  • offline access - online instructions for requesting the resource from the dataset
  • order - online order process for obtaining the dataset
  • search - online search interface for seeking out information about the dataset
