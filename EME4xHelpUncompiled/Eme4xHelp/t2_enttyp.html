EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Entity Type
Type: compound
Short Name: enttyp

FGDC Definition

The definition and description of a set into which similar entitiy instances are classified.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but is required if information is provided within the detailed description element. If information is provided in the detailed description element, users should document the entity label, definition, and definition source for each of the entities represented in the data set. This compound element provides the entity type label, definition, and definition source fields for a user to be able to document this information.
