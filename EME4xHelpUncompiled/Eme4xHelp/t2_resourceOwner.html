EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Resource Owner
Type: compound
Short Name: metc

FGDC Definition

The party responsible for the metadata information.

EPA Requirement/Guidance

Provide detailed information regarding who should be contacted regarding the metadata specifically. The contact person should be an individual who is knowledgable about the metadata, and may be able to answer questions about the metadata record, as opposed to the data specifically (this may also be the same person as the data set point of contact). Details on how to provide contact information for the data set are provided in Tab 1.

Domain

None.
