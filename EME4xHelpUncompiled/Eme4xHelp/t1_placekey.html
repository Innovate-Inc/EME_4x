EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - Place Keywords

Optional, Repeatable element

EPA Requirement/Guidance

This element is not required for minimum EPA compliancy but is encouraged to promote discovery of assets by location.

Note: User-defined keywords can be added to the user interface using the EME database. They are stored in the "KeywordsPlace" table. To add a new keyword to the EME database, follow these guidelines for Editing the Database.
