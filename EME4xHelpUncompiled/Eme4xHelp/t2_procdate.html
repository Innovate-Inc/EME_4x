Element: Process Date
Type: date
Short Name: procdate

FGDC Definition

The date when the event was completed.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but becomes required when the processing steps element is used.

The standard date format is yyyymmdd.
