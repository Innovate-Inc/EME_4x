Element: Source Currentness Reference

FGDC Definition

The basis on which the source time period of content information of the source data set is determined.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy. However, this element becomes mandatory if you choose to include the Source Information element in your record.
