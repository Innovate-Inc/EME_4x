EPA Metadata Editor (EME)

Interface Element Description: System of Records

Optional element

EPA Requirement/Guidance

The EPA's Privacy Act Systems of Records is temporarily being stored within this element. It is recognized that Data.gov defines this element as a URL, and ISO 19115 defines this element as a constraint on the fitness for use of the resource; however, EME 4.0 is currently using this field as a temporary placeholder for the system name.
