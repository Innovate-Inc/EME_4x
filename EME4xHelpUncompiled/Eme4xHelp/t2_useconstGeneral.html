EPA Metadata Editor (EME)

Interface Element Description: System of Records

Optional element

EPA Requirement/Guidance

The EPA's Privacy Act Systems of Records is temporarily being stored within this element. It is recognized that Data.gov defines this element as a URL, and ISO 19115 defines this element as a constraint on the fitness for use of the resource; however, EME 4.0 is currently using this field to store the system of record's URL. Once the user selects a system of record name, EME 4.0 will automatically convert the name to the associated URL in the saved XML file. For example, if the user selects "EPA-1: PeoplePlus", the output XML will look like:

We anticipate allowing additional instances of this element in future versions of EME.
