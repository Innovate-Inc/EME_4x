EPA Metadata Editor (EME)

Interface Overview and Element Description: Tab 2

Tab 2 of the EPA Metadata Editor allows the user to edit FGDC CSDGM Sections 6 (Distribution Information) and 7 (Metadata Information). The fields provided for the user to edit in Tab 2 correspond to all EPA required elements in Sections 6 and 7 of the FGDC CSDGM.

Geospatial metadata elements made available for editing within Tab 2 of the EPA Metadata Editor are described in greater detail below.

Tab 2 Elements Overview:

  • Distribution Information
    • Digital Transfer Options
    • Distribution Contact
      • Online Linkage (Make Part of Distribution)
    • Distribution Format
    • Standard Order Process
  • Dataset Constraints
    • Access Constraints
    • Distribution Liability
    • Security Classification
    • Use Constraints
  • Contacts
    • Metadata Author(s)
    • Resource Owner(s)
    • Resource Producer(s)
  • Metadata Information
    • Metadata Date
    • Metadata Future Review Date
    • Metadata Standard Name
    • Metadata Standard Version
    • Type of Resource
