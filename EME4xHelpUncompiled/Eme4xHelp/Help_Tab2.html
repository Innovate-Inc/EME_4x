EPA Metadata Editor (EME)

Interface Overview and Element Description: Tab 2

Tab 2 of the EPA Metadata Editor allows the user to edit FGDC CSDGM Sections 2 (Data Quality), 4 (Coordinate System Information) and 5 (Entity and Attribute Information). The fields provided for the user to edit in Tab 2 correspond to all EPA required elements in Sections 2 and 4 of the FGDC CSDGM, while also allowing users to edit some mandatory if applicable elements and sections (e.g., Section 5: 'Attributes' is not required).

Geospatial metadata elements made available for editing within Tab 2 of the EPA Metadata Editor are described in greater detail below.

Tab 2 Elements Overview:

  • Attribute Citation
  • Attribute Overview
  • Completeness
  • Horizontal Datum Name
  • Geographic Coordinate Units
  • Horizontal Coordinate System
  • Horizontal Positional Accuracy Report
  • Horizontal Positional Accuracy Tests Used
  • Horizontal Positional Accuracy Value
  • Logical Consistency Report
  • Overview Description
  • Processing Steps
  • Vertical Positional Accuracy Report
  • Vertical Positional Accuracy Value
  • Vertical Positional Accuracy Value Tests Used
  • State Plane Coordinate System Zone Identifier
