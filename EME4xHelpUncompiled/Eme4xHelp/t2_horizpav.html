EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Horizontal Positional Accuracy Value
Type: real
Short Name: horizpav

FGDC Definition

An estimate of the accuracy of the horizontal coordinate measurements in the data set expressed in (ground) meters.

EPA Requirement/Guidance

This element is not required for minimum EPA compliancy. If an estimate may not be provided, users may choose not to include this element.
