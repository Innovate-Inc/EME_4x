EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - EPA Theme Keywords

Optional, Repeatable element

EPA Requirement/Guidance

This field is not required for minimum EPA compliancy, but is recommended for EPA personnel and contractors in order to promote metadata consistency across the Agency. Select one or more EPA keywords (listed below) to describe your data resource where applicable. EPA Keywords should be used to describe EPA-produced assets.

Note: User-defined keywords can be added to the user interface using the EME database. They are stored in the "KeywordsEPA" table. To add a new keyword to the EME database, follow these guidelines for Editing the Database.
