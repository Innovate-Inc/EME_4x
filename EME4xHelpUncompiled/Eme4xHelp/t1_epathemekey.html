EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: EPA Theme Keyword
Type: theme
ISO Element Name: descriptiveKeywords

EPA Requirement/Guidance

This field is not required for minimum EPA compliancy, but is recommended for EPA personnel and contractors in order to promote metadata consistency across the Agency. Select one or more EPA keywords (listed below) to describe your data set where applicable. EPA Keywords should be used to describe EPA-produced geospatial assets.

Data.gov Requirement/Guidance

Mandatory Tags (or keywords) help users discover your resource; please include terms that would be used by technical and non-technical users.

ISO Definition

Theme keyword identifies a particular subject or topic.
