Interface Element Description: Tab 2

Element: VPF Terms Description
Type: compound
Short Name: vpfterm

FGDC Definition

Point and vector object information using the terminology and concepts from Department of Defense, 1992, Vector Product Format (MIL-STD-600006): Philadelphia, Department of Defense, Defense Printing Service Detachment Office. (Note that this reference to the VPF is used ONLY to provide a set of terminology for the point and vector objects.)

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but becomes required when the parent element, "Point and Vector Object Information" is used.
