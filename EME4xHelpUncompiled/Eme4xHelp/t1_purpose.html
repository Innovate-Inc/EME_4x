EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - Purpose

Optional element

EPA Requirement/Guidance

Brief description of the purpose for which the resource was originally created. This is a legacy field that was previously required by the FGDC. Although it is currently an optional field, this section helps establish boundaries of usage for the resource so that it will not be used for invalid purposes, and provides a historical context for which the resource was developed.

Example:
"This data resource was created to provide information about Toxic Release Inventory (TRI) locations in EPA Region 4 and are intended for geographic display and analysis at regional, state, and local levels."
