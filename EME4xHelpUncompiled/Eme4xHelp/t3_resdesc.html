EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Resource Description
Type: text
Short Name: resdec

FGDC Definition

The identifier by which the distributor knows the data set.

EPA Requirement/Guidance

This element is required to ensure that information is classified according to content types provided by Geospatial One-Stop (GOS). GOS uses a list of defined content types to organize geospatial resources for presentation to users. Users are required to classify information as one of the types listed below. The content type classification is based on access to the data set.

For example, if data are made available as a service (image or feature service), the classification should be listed as 'Live Data and Maps'. If data are made available through an ftp site or a downloadable location such as a website, the classification would be 'Downloadable Data'. If data are not accessible through the internet, they are considered 'Offline'.

The most common classifications used to document this element are 'Live Data and Maps', 'Downloadable Data', 'Applications', and 'Offline Data'. This classification should correspond with information provided in the online linkage element. For example, if your data are made available as a service, and are classified as Live Data and Maps, the online linkage should be documented appropriately to reference a 'Live Data and Service' type of URL. See the description provided for the online linkage field for more information on how to provide online linkage URLs.
