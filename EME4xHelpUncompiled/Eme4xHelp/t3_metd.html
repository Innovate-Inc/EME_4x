EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Metadata Date
Type: date
Short Name: metd

FGDC Definition

The date that the metadata were created or last updated.

EPA Requirement/Guidance

This field is required for minimum EPA compliancy as per FGDC requirements. Rely on FGDC definition for the metadata date field and follow date domain format restrictions as posted below.

Domain

free date
This date should be documented in YYYY, YYYMM, or YYYMMDDD format, with Y=year, M=month, D=day. No slashes are permitted.
