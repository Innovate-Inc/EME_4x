EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Distribution Contact
Type: compound
Short Name: distinfo

FGDC Definition

The party from whom the data set may be obtained.

EPA Requirement/Guidance

Provide detailed information in this field to identify the person or organization that should be contacted regarding accessing the data or asset. Contact information within this element may or may not be the same as that listed for the contact for the data and/or metadata.

Data.gov Requirement/Guidance

The EME Distribution Contact element includes 3 mandatory Data.gov elements:
  • Publisher - The plaintext name of the entity publishing this resource, corresponding to the FGDC contact organization.
  • Contact Name - A contact person's name for the asset, corresponding to the FGDC contact person.
  • Contact Email - A valid email address for the contact person listed above. An email address corresponding to a helpdesk or distribution list is an acceptable alternative.
It does not matter whether the "Primary Person" or "Primary Organization" radio button is selected, as long as the metadata contains all three elements above. If validation fails, the contact info may need to be edited in the EME database or by opening up the metadata document in a text editor.

Domain
