EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Contacts (Overview)
Type: compound CI_ResponsibleParty
ISO Element Name: CI_ResponsibleParty

EPA Requirement/Guidance

Provide detailed information in this component to identify the person or organization that should be contacted regarding the resource. Contact information within this element may or may not be the same in each section and can be repeated within each section. Currently, contact information is used with the following elements:
  • Distribution Contact - The party from whom the data set may be obtained.
  • Metadata Author - The metadata author(s) responsible for the metadata information.
  • Resource Owner - The owner(s) of the resource
  • Resource Producer - Those who created/produced the resource
These compound elements use an ISO CI_Responsible Party component. The EME Contacts_Info Table stores most of the items required to populate a section. To pass validation a contact in the Contacts_Info table should contain at the minimum the Contact Name, Organization, and Email. Additionally, for each contact used within a metadata record the user will need to choose a Role Code (required) and Program Code (if applicable).

Data.gov Requirement/Guidance

The EME Contact elements includes 3 mandatory Data.gov elements:
  • Organization - The plaintext name of the entity publishing this resource, corresponding to the FGDC contact organization.
  • Contact Name - A contact person's name for the asset, corresponding to the FGDC contact person.
  • Contact Email - A valid email address for the contact person listed above. An email address corresponding to a helpdesk or distribution list is an acceptable alternative.
If validation fails, the contact info may need to be edited in the EME database or by editing it direclty in the user interface.

ISO Definition

The identification of those responsible for the resource and their role in the resource.

FGDC Definition

The identification of those responsible for the resource and their role in the resource.
