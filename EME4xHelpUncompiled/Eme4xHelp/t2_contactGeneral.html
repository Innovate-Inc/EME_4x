EPA Metadata Editor (EME)

Interface Element Description: Tab 2 - Contacts (Overview)

Mandatory element

EPA Requirement/Guidance

This section provides detailed information on the person or organization that should be contacted regarding the resource. Contact information within this element may or may not be the same in each section and can be repeated within each section. At a minimum, one Metadata Author and one Resource Owner are required.

The required elements for Resource Owner (if added), Resource Producer (if added), and Metadata Author are: Role, Organization, and E-mail.

To add a contact, follow these guidelines on: How to Add a Contact

Contacts Section:

  • Resource Owner - The owner(s) of the resource. This element is mandatory per Data.gov requirements.
  • Resource Producer - Those who created/produced the resource.
  • Metadata Author - The metadata author(s) responsible for the metadata information. This element is mandatory per ISO requirements

Note: the Distribution Section also contains a potential Contact. A Distribution section is not mandatory, but if one is present, a Distribution Contact is required. More information on that element is available at the link below.

  • Distribution Contact - person or organization that should be contacted regarding accessing the resource. This element is mandatory if applicable per EPA requirements
