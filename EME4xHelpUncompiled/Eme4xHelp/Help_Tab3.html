EPA Metadata Editor (EME)

Interface Overview and Element Description: Tab 3

Tab 3 of the EPA Metadata Editor allows the user to edit FGDC CSDGM Sections 6 (Distribution Information) and 7 (Metadata Information). The fields provided for the user to edit in Tab 3 correspond to all EPA required elements in Sections 6 and 7 of the FGDC CSDGM.

Geospatial metadata elements made available for editing within Tab 3 of the EPA Metadata Editor are described in greater detail below.

Tab 3 Elements Overview:

  • Distribution Contact
  • Distribution Information
  • Distribution Liability
  • Metadata Contact
  • Metadata Date
  • Metadata Future Review Date
  • Metadata Information
  • Metadata Standard Name
  • Resource Description
