Interface Element Description: Tab 2

Element: Vertical Positional Accuracy Value
Type: real
Short Name: vertaccv

FGDC Definition

An estimate of the accuracy of the vertical coordinate measurements in the data set expressed in (ground) meters.

EPA Requirement/Guidance

This element is not required for minimum EPA compliancy. If vertical data can be tested for uncertainty or accuracy assessments, this field should be used to denote the estimate of the accuracy of the vertical measurement.
