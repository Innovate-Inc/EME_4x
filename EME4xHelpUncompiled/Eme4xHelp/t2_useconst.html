EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Use Constraints
Type: text
Short Name: useconst

FGDC Definition

Restrictions and legal prerequisites for using the data set after access is granted.

EPA Requirement/Guidance

For data developed by other parties, retain use constraints information, and add the EPA General Use Constraints Statement (shown below). For data developed in-house, determine scale, accuracy, and other limitations related to using the data. If there are no known use constraints for the data set, use the term 'None' and provide the EPA General Use Constraints Statement (shown below).

Domain

EPA General Use Constraints Statement: "Please check sources, scale, accuracy, currentness and other available information. Please confirm that you are using the most recent copy of both data and metadata."
