EPA Metadata Editor (EME)

Interface Element Description: Tab 2 - Distribution Contact

Mandatory element

EPA Requirement/Guidance

Provide detailed information in this field to identify the person or organization that should be contacted regarding accessing the data resource. Contact information within this element may or may not be the same as that listed for other sections. The minimum requirements for this element are: Role, Organization, and E-mail.

Note: If an incoming metadata record previously used distributionInfo/MD_Distribution/distributorFormat or distributionInfo/MD_Distribution/transferOptions, the content in those elements will be moved to a new record that says, "Content Missing, Add A Contact" in the "Individual" element.

To add a contact, follow these guidelines on: How to Add a Contact
