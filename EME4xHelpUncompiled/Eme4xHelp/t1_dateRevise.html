EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Revised Date
Type: date
ISO Element Name: gco:Date

EPA Requirement/Guidance

This element is required for EPA geospatial assets as per minimum FGDC standards. Follow FGDC guidance and information provided below for implementation of this field.

Data.gov Requirement/Guidance

Mandatory Most recent date on which the dataset was changed, updated or modified.

ISO Definition

Identifies when the resource was examined or re-examined and improved or amended.

FGDC Definition

The year, month, and day the resource was revised.

Domain

  • YYYY-MM-DD
