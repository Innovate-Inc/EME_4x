EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Metadata Standard Name
Type: text
Short Name: metstdn

FGDC Definition

The name of the metadata standard used to document the data set.

EPA Requirement/Guidance

Required for EPA geospatial assets as per minimum FGDC standards. Typically, the value for this field will be 'FGDC Content Standard for Digital Geospatial Metadata'. Follow FGDC guidance for implementation of this field.
