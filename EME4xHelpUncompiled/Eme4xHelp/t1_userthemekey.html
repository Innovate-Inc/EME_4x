EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: User Theme Keyword
Type: theme
ISO Element Name: descriptiveKeywords

EPA Requirement/Guidance

This field is not required for minimum EPA compliancy, but is made available for users who wish to enter keywords that are not part of the ISO or EPA lists.

Note: User-defined keywords are added to the user interface using the EME database. They are stored in the "KeywordsUser" table. To enter new keywords, open the "KeywordsUser" table and add the new keywords.

Data.gov Requirement/Guidance

Optinal Tags (or keywords) help users discover your resource; please include terms that would be used by technical and non-technical users.

ISO Definition

Theme keyword identifies a particular subject or topic.
