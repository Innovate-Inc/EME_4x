EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - User Theme Keywords

Optional, Repeatable element

EPA Requirement/Guidance

This field is not required for minimum EPA compliancy, but is made available for users who wish to enter keywords that are not part of the ISO or EPA lists.

Note: User-defined keywords can be added to the user interface using the EME database. They are stored in the "KeywordsUser" table. To add a new keyword to the EME database, follow these guidelines for Editing the Database.
