EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - Bounding Box Description

Optional element

EPA Requirement/Guidance

This element should be used to provide a brief one- or two-word description of the geographic location of the data resource. This may be one of the EPA regions available in the drop-down list, political unit (country, state name, state acronym, county, city) or a physical place name (e.g. "Cooley Landing" or "Camp Minden"). This element is optional; however, the east, west, north and south bounding coordinates are required elements.

Note: Bounding box descriptions (and bounding coordinates) can be added to the EME database. They are stored in the "BoundingBox" table. To add a new bounding box description to the EME database, follow these guidelines for Editing the Database.
