EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Source Information

Type: compound

Short Name: srcinfo

FGDC Definition

List of sources and a short discussion of the information contributed by each.

EPA Requirement/Guidance

This element is not required for minimum EPA compliancy. However, when this element is used, a number of additional child elements become required. Please note that if you choose to use the source information element in your record, a number of additional fields will become required within your metadata record.

Source Information includes the following child elements:

  • Source Citation -- Reference for a source data set
  • Source Scale Denominator -- Denominator of the representative fraction on a map (e.g. 24,000 on a 1:24,000 map)
  • Type of Source Media -- The medium of the source data set)
  • Source Time Period of Content -- Time period(s) for which the source data corresponds to the ground
  • Source Citation Abbreviation -- Short-form alias for the source citation)=
  • Source Contribution -- Brief statement identifying the information contributed by the source to the data set
