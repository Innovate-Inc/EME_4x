EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element:Unrepresentable Domain
Type: free text
Short Name: udom

FGDC Definition

Description of the values and reasons why they cannot be represented.

EPA Requirement/Guidance

The unenumerated domain should be used if the attribute being documented does not contain information that could be represented in a standard list or set of known values (e.g., the values for an attribute of people's names). In this case, provide free text to describe the attribute and the information that it may contain.
