EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Progress
Type: text
Short Name: progress

FGDC Definition

The state of the data set.

EPA Requirement/Guidance

Provide information (in general) about the status of the data set (e.g., is it complete, in work, or planned?). This element is required for EPA geospatial assets as per minimum FGDC standards. Follow FGDC guidance for implementation of this field. Note that the use of one of the three values shown in the domain list below is required for compliancy. Users may not enter free text into this field.

Domain

  • Complete
  • In work
  • Planned
