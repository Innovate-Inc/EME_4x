EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Horizontal Datum Name
Type: text
Short Name: horizdn

FGDC Definition

The identification given to the reference system used for defining the coordinates of points.

EPA Requirement/Guidance

This element is required in order to adequately portray the geodetic datum.

Domain
