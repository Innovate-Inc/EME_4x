EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Overview Description
Type: compound
Short Name: overview

FGDC Definition

Summary of, and citation to detailed description of, the information content of the data set.

EPA Requirement/Guidance

This element is not required for EPA compliancy but is recommended for data sets that contain attribute information. The overview section may be used to provide a general overview of information about a data set's attributes. The overview information should be provided in the Entity and Attribute Overview section. The overview reference information should be included in the Entity and Attribute Detail Citation field.
