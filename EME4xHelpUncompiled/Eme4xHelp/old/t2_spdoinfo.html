EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Spatial Data Organization Information
Type: compound
Short Name: spdoinfo

FGDC Definition

The mechanism used to represent spatial information in the data set.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy. It is recommended that users document Spatial Data Organization Information for their data sets where applicable. The Spatial Data Organization Information fields describe the methods you used to represent spatial information in the data set, and how the data are organized, including direct and indirect spatial references. Use these fields to document the number and type of objects in your data set, the terminology set you use to describe the data, and other information about the nature of the data.
