EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Publication Place
Type: text
Short Name: pubplace

FGDC Definition

The name of the city and state where the data set was published or released.

EPA Requirement/Guidance

This element is required for EPA geospatial assets in order to comply with publishing requirements for Geospatial One Stop (GOS) and ESRI catalog implementations. Provide information on the location where the information was published (city, state).
