EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Metadata Future Review Date
Type: date
Short Name: metfrd

FGDC Definition

The date by which the metadata entry should be reviewed.

EPA Requirement/Guidance

This field is required for minimum EPA compliancy. It is not required for minimum FGDC compliancy. EPA Guidance indicates that this date should be equal to the metadata date + 4 years at a maximum. Follow date format restrictions listed below for proper date formatting.

Domain

free date
This date should be documented in YYYY, YYYMM, or YYYMMDDD format, with Y=year, M=month, D=day. No slashes are permitted.
