EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Entity and Attribute Detail Citation
Type: text
Short Name: eadetcit

FGDC Definition

Reference to the complete description of the entity types, attributes, and attribute values for the data set.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but is required if information is provided within the attribute overview element. If information is provided in the attribute overview section, users should document the source materials or citation information within this element. This may be in the form of literature cited, a URL, or references to other sources.
