EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Publisher
Type: text
Short Name: publish

FGDC Definition

The name of the individual or organization that published the data set.

EPA Requirement/Guidance

This element is required for EPA geospatial assets in order to comply with publishing requirements for Geospatial One Stop (GOS) and ESRI catalog implementations. Provide the name of the organization responsible for publishing the data set/resource.
