EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Horizontal Coordinate System
Type: compound
Short Name: horizsys

FGDC Definition

The reference frame or system from which linear or angular quantities are measured and assigned to the position that a point occupies.

EPA Requirement/Guidance

This element is required for minimum EPA compliancy to ensure that horizontal coordinate system information is adequately documented for the data set.
