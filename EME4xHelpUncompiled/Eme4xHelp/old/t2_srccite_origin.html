EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Source Citation: Originator
Type: text
Short Name: origin

FGDC Definition

The name of an organization or individual that developed the data set. If the name of editors or compilers are provided, the name must be followed by "(ed.)" or "(comp.)" respectively.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy. However, this element becomes required when the parent element, Source Citation Information, is included.
