EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Entity and Attribute Overview
Type: text
Short Name: eaover

FGDC Definition

Detailed summary of the information contained in a data set.

EPA Requirement/Guidance

This element is not required for EPA compliancy but is recommended for data sets that contain attribute information. This element is used to provide an overview of attribute information as opposed to the detailed attribtues section that lists individual attribtues one by one. Both options may be used together. This element allows the user to provide a general overview of attribute information and include a link or reference to a published citation or data dictionary. The overview reference information should be included in the Entity and Attribute Detail Citation field.
