EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Source Time Period of Content
Type: compound
Short Name: srctime

FGDC Definition

The time period(s) for which the source data set corresponds to the ground.

EPA Requirement/Guidance

This element is required for minimum FGDC compliancy. However, this element becomes mandatory if you choose to include the Source Information element in your record.
