EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Completeness Report
Type: text
Short Name: complete

FGDC Definition

Information about omissions, selection criteria, generalization, definitions used, and other rules used to derive the data set.

EPA Requirement/Guidance

This element is required for EPA geospatial assets as per minimum FGDC standards within section 2. If possible, provide an assessment about the completeness of the data set (e.g., outlining any known omissions,etc.). Follow FGDC guidance for implementation of this field.
