EPA Metadata Editor (EME)

Interface Element Description: Tab 3

Element: Distribution Information
Type: compound
Short Name: distinfo

FGDC Definition

Information about the distributor of and options for obtaining the data set.

EPA Requirement/Guidance

This section is required by EPA to ensure inclusion of the resource description element (used by Geospatial One Stop and ESRI search tools) and to ensure proper documentation of distribution liability.

Domain

None.
