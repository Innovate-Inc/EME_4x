Interface Element Description: Tab 2 - Distribution Format

Optional element

EPA Requirement/Guidance

This element describes the computer language construct that specifies the representation of data objects in a record, file, message, storage device, or transmission channel. The following elements are included in this section:

Name - Name of the data transfer format.

Version - Version of the format (date, number, etc).

Amendment Number - Format version amendment number.

Specification - The subset name, profile, or product specification of the format.

File Decompression Technique - Description of recommended processes or algorithms to apply to the compressed resource.
