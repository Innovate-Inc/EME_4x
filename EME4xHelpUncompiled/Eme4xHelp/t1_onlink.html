EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Online Linkage
Type: text
Short Name: onlink

FGDC Definition

The name of an online computer resource that contains the data set.

EPA Requirement/Guidance

Online linkage is not required for EPA minimum requirements. It is highly encouraged in order to promote reuse of and access to resources.

This element should reflect the 'Data Resource Type' classification provided in Tab 3. For example, if the resource type is listed as 'Live Data and Services', the URL provided in this field should reflect a live data and service URL. An example of how to document live data and services is provided below. For more information on resource types, see the 'Data Resource Type' field in Tab 3.
In general, the inclusion of two online linkage URLs is recommended.
  • Primary Linkage: the primary location for accessing the data, application, or service.
  • Secondary Linkage: a website or other location where more information can be obtained about the asset.

Domain

Examples:
Image Services (WMS Connector):
"http://edg.epa.gov/wmsconnector/com.esri.wms.Esrimap/WasteWaterImage?request=getcapabilities&service=WMS&version=1.1.1"
Feature Services (WFS Connector):
"http://edg.epa.gov/wfsconnector/com.esri.wfs.Esrimap/NPL_FS?request=getcapabilities&service=wfs"
