EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Source Scale Denominator
Type: integer
Short Name: srcscale

FGDC Definition

The denominator of the representative fraction on a map (for example, on a 1:24,000-scale map, the Source Scale Denominator is 24000).

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy.

Domain

Source Scale Denominator > 1
