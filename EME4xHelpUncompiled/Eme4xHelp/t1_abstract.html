EPA Metadata Editor (EME)

Interface Element Description: Tab 1

Element: Abstract
Type: text
Short Name: abstract

FGDC Definition

A brief narrative summary of the data set.

EPA Requirement/Guidance

This field is required for EPA geospatial assets as per minimum FGDC standards. Typically, this field should be used to provide a general overview about the asset. Follow FGDC guidance for implementation of this field.

Data.gov Requirement/Guidance

Mandatory human-readable description (e.g., an abstract) with sufficient detail to enable a user to quickly understand whether the asset is of interest.

Domain

free text
