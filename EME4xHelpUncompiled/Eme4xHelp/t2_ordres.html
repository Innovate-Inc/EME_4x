EPA Metadata Editor (EME)

Interface Element Description: Tab 2

Element: Ordinate Resolution
Type: real
Short Name: ordres

FGDC Definition

Ordinate Resolution: The (nominal) minimum distance between the "y" or row values of two adjacent points, expressed in Planar Distance Units of measure.

EPA Requirement/Guidance

This element is not required for minimum FGDC compliancy, but becomes required when the "Planar Coordinate Information" element is used. Note: The Y field in EME corresponds the Ordinate Resolution FGDC element.

These values represent the minimum difference between Y (ordinate) values in the planar data set. For raster data, the values normally equal pixel size, e.g. 30. For vector data, the values usually indicate the 'fuzzy tolerance' or 'clustering' setting that establishes the minimum distance at which two points will NOT be automatically converged by the data collection device (digitizer, GPS, etc.). NOTE: units of measures are provided under element Planar Distance Units (e.g. meters or milimeters).

This information is usually captured when one uses the EPA Synchronizer to synchronize the coordinate system information from the data set. When one uses the EPA Synchronizer, the absres and ordres fields will be automatically populated with the correct information from the dataset. The EPA Synchronizer is the best way to capture this information, and it is recommended where possible for documenting this element. One can attempt to do this calculation manually however, using ArcMap or another GIS program. Calculating this information in ArcMap may require a few steps. The ordres value represents the minimum possible difference between coordinate values. One will need to perform a function that compares all of the points or vertices within the data set and then determine the minimum distance among all of them. This is best done using a function such as the "near" function (ArcPy) or another spatial processing function.

One may also reference the ESRI help to review the "Resolution" property of a geodatabase feature class.

The "Ten Most Common Metadata Errors" document at the FGDC website provides additional information regarding this element.

Domain

Ordinate Resolution > 0.0

Longitude Resolution > 0.0
