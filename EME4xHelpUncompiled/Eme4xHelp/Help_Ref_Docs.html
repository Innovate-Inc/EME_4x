Reference Information

Data.gov

Data.gov is a searchable catalog of federal data that provides access to "raw" datasets and various tools. In the "raw" data catalog, you may access data in XML, Text/CSV, KML/KMZ, Feeds, XLS, or ESRI Shapefile formats.

EPA Environmental Dataset Gateway

The EPA Environmental Dataset Gateway (EDG) is the U.S. Environmental Protection Agency’s (EPA) central access point for geospatial resources and datasets on the Web. OEI manages the EDG in partnership with data stewards within the regions and program offices.

EPA Geospatial Metadata Technical Specification

EPA has developed Agency guidelines for publishing geospatial metadata for data sets, applications, and services. For more information about the EPA Geospatial Metadata Technical Specification, please visit http://epa.gov/geospatial/policies.html.

FGDC Content Standard for Digital Geospatial Metadata (CSDGM)

For additional information on the Federal Geographic Data Committee's (FGDC) Content Standard for Digital Geospatial Metadata (CSDGM), please visit FGDC CSDGM.

Innovate!, Inc.

Innovate!, Inc. is an 8(a) company that provides geospatial services for federal, state, and private clients. For more information, please visit our website at http://innovateteam.com.
