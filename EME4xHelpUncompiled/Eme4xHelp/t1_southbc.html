Interface Element Description: Tab 1

Element: South Bounding Coordinate
Type: real
ISO Element Name: southBoundLatitude

EPA Requirement/Guidance

Required for EPA geospatial assets as per minimum FGDC standards. Follow FGDC guidance for implementation of this field.

Data.gov Requirement/Guidance

NA

ISO Definition

Southernmost coordinate of the limit of the resource extent (EX_GeographicBoundingBox), expressed in longitude in decimal degrees referenced to WGS 84.

FGDC Definition

Southern-most coordinate of the limit of coverage expressed in latitude.

Domain

-90.0 <= South Bounding Coordinate <= 90.0
