EPA Metadata Editor (EME)

Interface Element Description: Tab 1 - ISO Theme Keywords

Mandatory, Repeatable element

EPA Requirement/Guidance

Inclusion of one or more ISO keywords is required for minimum EPA compliancy which supports inter-Agency sharing initiatives. The ISO 19115 Topic Category is a general categorization of data resources that is intended to provide data classification consistency across agencies. Select one or more of the basic ISO keywords for a general classification of your data or application.

Note: User-defined keywords can be added to the user interface using the EME database. They are stored in the "KeywordsISO" table. To add a new keyword to the EME database, follow these guidelines for Editing the Database.
