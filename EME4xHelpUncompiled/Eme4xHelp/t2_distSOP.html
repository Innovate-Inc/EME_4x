Interface Element Description: Tab 2 - Standard Order Process

Optional element

EPA Requirement/Guidance

This section describes the process by which the resource was obtained or received and other related instructions or fee information. The following elements are included in this section:

Fees - Fees and terms for obtaining the data resource. Best Practices: Include at the end the monetary unit alphabetic code as specified in ISO 4217. Common codes: United States = USD, Canada = CAD, Mexico = MXN or MXV. Example: 45.00USD

Available Date - Date and time the resource will be available.

Ordering Instructions - General instructions, terms, and services provided by the data distributor.

Turnaround - Typical time required for filing a data request.
