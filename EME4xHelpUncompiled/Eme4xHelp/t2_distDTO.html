EPA Requirement/Guidance

This section provides information about the technical means and media by which a resource is obtained from the distributor. The following elements are included in this section:
  • Units Of Distribution - Tiles, layers, geographic areas, etc. in which the data is available.
  • Transfer Size - Estimated size of the transfer unit in the specified format, expressed in megabytes. Best Practices: Value greater that 0.0.
  • On Line - Information about the online sources where the data/dataset may be obtained. ISO Element Name: CI_OnlineResource
    • Linkage - Internet location (address) for on-line access which uses a Uniform Resource Locator address or similar addressing scheme. NOTE: [WE NEED TO INCLUDE THE OLD ONLINE LINKAGE HELP HERE. See Onlink Page
    • Function - Code for function performed by the online resource (CI_OnlineFunctionCode)
    • Protocol - The connection protocol to be used such as http, ftp, etc.
    • Application Profile - Name of an application profile that can be used with the online resource.
    • Name - Name of the online resource.
    • Description - Description of the online resource that provides the resource sought.
  • Off Line - Information about the offline distribution media regarding the name, density units, volumes, medium format, and medium note used to describe the transfer of data to a medium. ISO Element Name: MD_Medium
    • Name - Medium name code. Type: MD_MediumNameCode Domain: cdRom, dvd, dvdRom, 3halfInchFloppy, 5quarterInchFloppy, 7trackTape, 9trackTape, 3480Cartridge, 3490Catridge, 3580Cartridge, 4mmCartridgeTape, 8mmCartridgeTape, 1quarterInchCartridgeTape, digitalLinearTape, onLine, satellite, telephoneLink, hardcopy.
    • Density - The recording density on the specified media. Best Practices: Value greater than 0.0.
    • Density Units - The recording density units. Best Practices: densityUnits is mandatory when density is documented.
    • Volumes - Identification of the number of recorded items on the media. Type: gco:Integer, Domain: any whole number, Best Practices: Value greater than 0.
    • Medium Format - Method used to write to the medium. Type: MD_MediumFormatCode, Domain: cpio, tar, highSierra, iso9660, iso9660RockRidge, iso9660AppleHFS.
    • Medium Note - Description of limitations or requirements for using the medium.
