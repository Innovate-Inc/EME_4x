EPA Metadata Editor (EME)

Interface Element Description: Tab 2 - Digital Transfer Options

Optional element; however, if this element is included, "Linkage" and "Function" are required elements.

EPA Requirement/Guidance

This section provides information about the technical means and media by which a resource is obtained from the distributor. There are two resource options: online or offline. The following elements are included in this section:

Units Of Distribution - Tiles, layers, shapefiles, raster files, geographic areas, etc. in which the resource is available.

Transfer Size - Estimated size of the transfer unit in the specified format, expressed in megabytes. Best Practices: Value greater than 0.0.

Online Resource - Information about the online resource where the data resource may be obtained.

  • Linkage - Internet location (address) for on-line access which uses a Uniform Resource Locator (URL) address or similar addressing scheme. This is a required element.
  • Function - Function performed by the online resource. This is a required element.
  • Protocol - The connection protocol to be used, e.g. http, ftp, etc.
  • Application Profile - Name of an application profile that can be used with the online resource, e.g. Dublin Core. Designed to provide information about applications that might be able to use the link automatically
  • Name - Name of the online resource, e.g. NODC Ocean Archive System
  • Description - Description of the online resource that provides the resource sought, e.g. "This is a documentation-only accession to contain the various web pages, directories, and documents providing information to the responders to the Deepwater Horizon event."

Offline Resource - Information about the offline distribution media regarding the medium name, density, density units, volumes, medium format, and medium note used to describe the transfer of data to a medium.

  • Medium Name - Medium type, e.g. cdRom, dvd, dvdRom, 3halfInchFloppy, 5quarterInchFloppy, 7trackTape, 9trackTape, 3480Cartridge, 3490Catridge, 3580Cartridge, 4mmCartridgeTape, 8mmCartridgeTape, 1quarterInchCartridgeTape, digitalLinearTape, onLine, satellite, telephoneLink, or hardcopy.
  • Density - The recording density on the specified media. Best Practices: Value greater than 0.0.
  • Density Units - The recording density units. Best Practices: Density Units is mandatory when density is documented.
  • Volumes - Identification of the number of recorded items on the media. Best Practices: Value greater than 0.
  • Medium Format - Method used to write to the medium, e.g. cpio, tar, highSierra, iso9660, iso9660RockRidge, or iso9660AppleHFScdRom.
  • Medium Note - Description of limitations or requirements for using the medium.

Domain

Function:

  • download - online instructions for transferrring data from one storage device or system to another
  • information - online information about the dataset
  • offline access - online instructions for requesting the resource from the dataset
  • order - online order process for obtaining the dataset
  • search - online search interface for seeking out information about the dataset
